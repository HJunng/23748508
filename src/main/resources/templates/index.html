프로젝트 개요

본 프로젝트는 Spring Boot와 JPA를 활용하여 사용자 인증, 게시글 CRUD, 페이징 처리 등을 포함한 기능을 개발했습니다. 주요 목표는 안정적이고 확장 가능한 백엔드 시스템을 구현하는 것이며, Thymeleaf를 통해 서버에서 동적으로 HTML을 렌더링하여 사용자와 상호작용합니다. RESTful API 대신 서버 사이드 렌더링(Server-Side Rendering, SSR)을 기반으로 페이지를 제공하여 초기 로드 속도를 최적화하고 검색 엔진 친화적인 설계를 구현했습니다.

구현 과제

로그인

사용자가 아이디와 비밀번호를 입력하면 서버는 입력값을 데이터베이스와 비교해 인증합니다. 인증 성공 시 세션(Session)을 생성해 로그인 상태를 유지합니다.

  • Spring Security
  • H2
  • Thymeleaf
POST /auth/login

로그아웃

사용자가 로그아웃을 요청하면 서버는 세션 ID를 무효화하여 로그인 상태를 종료합니다.

  • Spring Security
  • Session Management
POST /auth/logout

게시판 목록

제목, 작성자 ID, 조회수, 첨부파일 여부, 작성일 등 게시글 정보를 페이징 처리하여 출력합니다. 키워드로 게시글 검색이 가능합니다.

  • Spring Boot
  • JPA
  • H2
GET /api/posts

게시판 보기

선택한 게시글의 제목, 작성자, 작성일, 조회수, 내용을 상세히 출력하며, 조회수가 증가합니다.

  • JPA
  • H2
GET /api/posts/{id}

게시판 작성

인증된 사용자만 게시글 작성이 가능하며, 제목과 내용을 입력하고 파일을 첨부할 수 있습니다.

  • Spring Security
  • MultipartFile
POST /api/posts/write

게시물 수정/삭제

본인이 작성한 게시글만 수정 및 삭제할 수 있습니다.

  • Spring Security
  • JPA
PUT /api/posts/{id}/edit DELETE /api/posts/{id}
